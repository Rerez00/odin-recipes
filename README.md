🍽️ Odin Recipes

Odin Recipes is a beginner-friendly web project created as part of The Odin Project curriculum. This project focuses on building a simple recipe website using HTML, CSS, and JavaScript to practice core web development fundamentals.

Through this project, I aim to:

Structure web pages using semantic HTML

Style layouts and improve visual design with CSS

Add basic interactivity and logic using JavaScript

Strengthen my understanding of how these technologies work together

This project serves as a foundation for my journey toward becoming a full-stack developer and will be improved as my skills grow.
